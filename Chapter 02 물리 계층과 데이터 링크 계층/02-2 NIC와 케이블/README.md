### Q1. 싱글 모드 광섬유 케이블과 멀티 모드 광섬유 케이블의 차이는?

<details>
<summary>정답</summary>

-싱글 모드 광섬유 케이블은 장파장의 빛을 사용하며 코어의 지름이 작아 빛의 이동 경로가 한개, 장거리 전송에 적합합니다.

-멀티 모드 광섬유 케이블은 단파장의 빛을 사용하고, 코어의 지름이 커 빛의 이동 경로가 여러개이고, 단거리 전송에 적합합니다.
